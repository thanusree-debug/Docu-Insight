# AI Document Explorer 🚀

AI Document Explorer is a Generative AI web application that allows users to upload PDF documents and ask questions using a Retrieval-Augmented Generation (RAG) pipeline.

## 🔥 Features

* Upload PDF documents
* Ask questions based on document content
* AI-generated answers
* Highlighted source text
* Suggested follow-up questions
* Interactive UI (Document + Chat + Insights)

## 🧠 Tech Stack

* Python
* Streamlit
* LangChain
* FAISS
* OpenAI API / LLM

## ⚙️ How It Works

1. Upload PDF
2. Convert text into embeddings
3. Store in vector database
4. Retrieve relevant context
5. Generate AI answer

## ▶️ Run Locally

pip install -r requirements.txt
streamlit run app.py
